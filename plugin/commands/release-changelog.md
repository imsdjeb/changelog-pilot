description: "Generate a human-readable changelog entry from commits since last release. Groups related changes and writes for end users."
argument-hint: "[version] e.g. 1.3.0"
---

Run the **CHANGELOG** workflow from the changelog-pilot skill. Analyze commits, generate a human-readable changelog entry in Keep a Changelog format, and offer to insert it into CHANGELOG.md.

If a version is provided, use it. Otherwise, use the recommended version from the preview.
